## ✨ Sobre
O **Miyano Bot** é um bot com vibe tranquila e organizada, inspirado no estilo do Miyano em **Horimiya**.  
Feito pra rodar de boa no **Termux**, com suporte a dependências essenciais como **Node.js** e **FFmpeg**.

**💡 Ideal para:** automações, comandos de utilidade, mídia, administração e diversão.

---

## ⚙️ Requisitos
- **Termux** atualizado
- **Node.js / Node LTS**
- **Git**
- **FFmpeg**
- Permissão de armazenamento no Android
